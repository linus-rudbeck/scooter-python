# scooter-python

## Installation & startup

1. Run `pip install ursina` or check here if you have problems with pip: https://stackoverflow.com/a/46757232/10043760
2. Run `py .\main.py`
3. Press <kbd>space</kbd> to jump

**Good luck, happy coding!**
